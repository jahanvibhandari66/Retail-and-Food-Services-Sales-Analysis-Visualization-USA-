Retail and Food Services Sales Analysis & Visualisation (USA)
1. Introduction
The Retail and Food Services Sales Analysis project aims to analyse consumer spending patterns in the United States using retail and food service sales data obtained from the U.S. government. The project focuses on transforming raw sales data into meaningful insights through SQL-based data management, analysis, and visualization techniques. By examining trends across different industries and time periods, the project helps understand market performance and consumer behavior.

2. Problem Statement
Large volumes of retail sales data are generated every month, making it difficult to manually identify trends and patterns. Businesses and analysts require efficient methods for storing, managing, and visualising this data to support decision-making.
This project addresses the challenge by creating a structured SQL database and interactive visualisations that allow users to analyse sales trends, compare industry performance, and identify seasonal patterns.

3. Objectives
Collect Retail and Food Services Sales data from U.S. government sources.
Design a relational database using SQL.
Clean and preprocess the dataset.
Analyse sales performance across different industries.
Identify growth trends and seasonal variations.
Create visual dashboards for easy interpretation of results.
